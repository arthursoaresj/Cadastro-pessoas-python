###### **Cadastro de Pessoas em Python**



Aplicação em Python e MySQL para cadastrar e pessoas e visualizar o banco de dado, uma melhoraia do meu outro projeto chamado "MenuPython".



###### **Funcionalidades**



* Cadastrar pessoas
* Vizualizar pessoas cadastradas
* Armazenamento das pessoas no MySQL
* Interface simples no terminal



###### **Estrutura**



cadastro-pessoas-python/

&nbsp;	SQL

&nbsp;	main.py

&nbsp;	README.md



###### **Como executar**

1. Instale as dependencias
2. Configure o Banco de dados
3. Execute:

&nbsp;	python main.py
